---
title: Cannot Access Documentation
description: Fix common access issues for Oleg Parashchuk docs.
---

## Symptoms

- Documentation site does not load
- Error 404 or forbidden access

## Possible Causes

- Incorrect URL
- Permissions not granted

## Solution

<Steps>
  <Step title="Check the URL">
    Ensure you used `https://docs.olegparashchuk.com` or your assigned docs link.
  </Step>
  <Step title="Login Required">
    Make sure you are logged in with the correct account.
  </Step>
  <Step title="Request Access">
    Email `support@olegparashchuk.com` for permission if needed.
  </Step>
</Steps>
