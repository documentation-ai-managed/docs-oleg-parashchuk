## Overview

Welcome to Oleg Parashchuk's documentation space. You can organize, manage, and share all your project documentation here with ease. This platform provides powerful tools to create structured docs, collaborate with teams, and keep everything up to date.

Use it to document APIs, user guides, internal wikis, or any project knowledge base. Start by creating pages, adding components, and customizing with your brand.

<Callout kind="info">
  Primary brand color: <code>#3B82F6</code>. Apply it consistently for buttons, links, and accents.
</Callout>

Get up and running in minutes.

## Brand Color Guidelines

Maintain consistency with the primary brand color <code>#3B82F6</code> (blue shade).

```css
:root {
  --brand-primary: #3B82F6;
  --brand-primary-hover: #2563EB;
}

.button {
  background-color: var(--brand-primary);
  color: white;
}
```

<Callout kind="tip">
  Use this color for primary actions. Avoid overuse on large areas to maintain readability.
</Callout>
